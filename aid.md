## РАБОТА С РЕПОЗИТОРИЕМ И КОММИТ

Чтобы Git начал отслеживать изменения в проекте, папку с файлами этого проекта нужно сделать Git репозиторием  Для этого команда ``` git init

Разгитить папку ``` rm -rf .git


Проверить состояние репозитория ``` git status

Создаём файлы работаем потом добовляем в репозиторий командой ``` Git add --all или ``` Git add .(добавит всю текущую папку )


Сохраняем файлы командой ``` Git commit -m (-m позволяет добавить к коммиту коментарий )


``` Git push отправить файлы на удалённый репозиторий
